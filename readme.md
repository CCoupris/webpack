# Webpack Template

## Features
- Basic files support (HTML, CSS, Stylus, JS, Images, Fonts)
- Extracted CSS
- Transpilled JS
- Sourcemap
- Clean Dist
- Dev server (with local network support)
- Static files
- Cache breaker

## Instructions

Install dependencies

```npm i```

Lunch dev server

```npm run dev```

Build for production

```npm run build```
